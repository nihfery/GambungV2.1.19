**Gamboeng Traceability** adalah aplikasi berbasis **React + Vite** yang dikembangkan untuk kebutuhan web modern dengan performa cepat dan struktur modular. 

Panduan berikut menjelaskan langkah-langkah sederhana untuk menjalankan project ini di lingkungan lokal.

## Langkah Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/nihfery/GamboengV1.0.git
2. **Install Semua Dependensi**
   ```bash
   npm install
3. **Jalankan Project dalam Mode Pengembangan**
   ```bash
   npm run dev
